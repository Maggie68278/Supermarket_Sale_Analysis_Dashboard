## Project Overview

This project presents an interactive **Supermarket Sales Analysis Dashboard** built using **Power BI**. The dashboard analyzes sales and profit performance across products, time, customer segments, and geographic regions in the United States.

The project workflow includes:
- Data extraction from a Kaggle supermarket dataset
- Data transformation
- Data modeling
- DAX calculations
- Interactive data visualizations

The dashboard provides actionable insights into the supermarket’s **sales trends, profitability, and financial performance**.

## Dashboard Visualizations

### KPI Cards
- Compare the supermarket’s **current-year financial performance** with its **previous year**.

### Bar Chart – Profit by Product
- Illustrates profit across different product categories.
- **Green** indicates profit.
- **Red** indicates loss.

### Line Chart – Sales Trend
- Displays the supermarket’s **sales revenue trend** from **2020 to 2023**.

### Shape Map – Profit by State
- Shows **profit distribution across U.S. states**.

### Segment Analysis
- Sales breakdown by customer segment:
  - Consumer
  - Corporate
  - Home Office

## Key Outcomes
